Project Overview

Chattea is a community-driven chat platform designed to connect coffee enthusiasts across local areas. The platform features location-based lobbies, coffee shop discovery tools, and real-time chat functionality that brings people together around their shared love of coffee culture.

Built with a warm, inviting aesthetic using coffee browns (#D2691E), cream tones (#F4E4C1), and rich dark roast colors (#8B4513), the design creates a cozy digital coffeehouse atmosphere that encourages community interaction and discovery.

Platform Features

Location-Based Lobbies

Chat rooms organized by local areas including Southport, Broadbeach, and Surfers Paradise. Each lobby shows current active users and capacity to help connect nearby coffee lovers.

Coffee Shop Discovery

Built-in locator map feature (in development) that will help users discover local coffee shops, read reviews, and plan coffee meetups with community members.

Community Profiles

User profiles with customizable settings, about sections, and help center integration to build meaningful connections within the coffee community.

Live Chat Features

Real-time messaging system with live servers (coming soon) and game mode chat rooms for interactive community experiences and coffee-themed discussions.
