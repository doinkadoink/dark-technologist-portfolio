ChatTea

2025 · HTML, CSS & JavaScript · Portfolio web piece

WHAT IT IS

A coffee-themed community prototype: themed lobbies, profile-style pages, and a warm visual identity built with semantic HTML, layout CSS, and light interactivity — part of my 2025 portfolio-site work alongside this main site.

The live class / demo build is hosted separately so tutors and visitors can click through the full flow.

Some items in the original pitch (e.g. live server chat, map locator) were marked as roadmap; the hosted version reflects what was submitted for the digital-design milestone.

FEATURES

  • Area-style lobbies (e.g. local Gold Coast naming) to organise conversation by place
  • Community profiles and help-style pages for a small social app feel
  • Responsive layout and readable type for phone-first review
  • Warm brand palette (caramel / cream on deep roast background) kept deliberately soft — no busy pattern layer
